배운 것 (Learned)

  • color-scheme는 스크롤바뿐 아니라 인쇄 캔버스 배경을 결정한다. 스크롤바 라이트화로 넣은 color-scheme가 헤드리스(OS 다크)에서 PDF 페이지 여백을 검게 만든 회귀 → @media print { color-scheme: light }로 차단. (화면 fix가 인쇄에 샌 사례)
  • 점진적 향상 애니메이션은 기본이 "보이는" 상태여야 한다. reveal을 opacity:0에서 시작했더니 PDF/no-JS에서 사라짐 → 인라인 width + CSS 키프레임(0→값), print는 강제 가시화.
  • 모바일 브레이크포인트는 인쇄에서도 켜진다. A4(~794px) < 980px라 PDF에서 카드가 1열로 쌓임(가독상 허용).
  • 시간민감 사실(LLM 가격·모델·SOTA)은 기억이 아니라 웹 1차 소스로. cutoff 이후 드리프트가 커서 6개 병렬 에이전트로 검증·인용.
  • 과한 인용 보수화는 진짜 인용을 지운다. 미래시점 arxiv ID를 할루시네이션으로 단정해 실재 시스템을 뺐다가, 검증 후 복원. 의심 시 "지우기" 전에 "검증"이 먼저.
